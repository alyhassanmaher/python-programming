# Tactical War Game — Python Programming Project

A turn-based tactical war game built with **Pygame**, featuring grid-based movement, multiple unit types, terrain, and a two-player local multiplayer mode.

## Gameplay

Two players take turns commanding armies on a grid battlefield. Each player controls 4 units and must eliminate all enemy units to win.

## Files

| File | Description |
|------|-------------|
| `final2-0.py` | Complete game source code |

## Features

- **4 unit types** with unique stats:
  - 🗡️ Infantry — balanced, high HP
  - 🏹 Archer — long attack range, fragile
  - 🐴 Cavalry — fast movement, high HP
  - 🛡️ Tank — heavy armor, highest HP and attack
- **Procedurally generated terrain** with obstacle tiles
- **Turn-based system** with a 2-moves-per-turn limit
- **Visual indicators** — health bars, move/attack highlights, selection outline
- **Game states** — Main Menu → Game → Game Over screen

## How to Run

```bash
pip install pygame
python final2-0.py
```

## Controls

| Action | Input |
